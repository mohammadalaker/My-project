# إضافة أصناف/مجموعات في الإكسل (مثل Tefal، Luminarc)

في ملف الإكسل **كشف القطع الصغيرة و المنزلي.xlsx**:

1. **عمود المجموعة/الصنف**  
   تأكد من وجود عمود لأسماء المجموعات أو الصنف. البرنامج يقرأ أي عمود بعنوان مثل:
   - المجموعة، Group، صنف، ماركة، Brand، الفئة، Category

2. **إضافة أصناف مثل Tefal و Luminarc**  
   في ذلك العمود اكتب في كل صف اسم الصنف/الماركة، مثلاً:
   - `Tefal` لمنتجات تيفال
   - `Luminarc` لمنتجات لومينارك
   - أو أي اسم آخر (مثل KMG Midea SDA، إلخ)

3. **بعد التعديل**  
   شغّل من المشروع:
   ```bash
   npm run read-excel
   ```
   ثم حدّث الصفحة في المتصفح؛ ستظهر المجموعات الجديدة (Tefal، Luminarc، إلخ) في **فهرسة المجموعات** ويمكنك الفلترة حسبها.
