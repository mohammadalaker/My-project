نظام التنبيهات الصوتية (Audio Manager — useSystemSounds)
========================================================
الملفات التالية تُحمّل مسبقاً لتقليل التأخير عند الاستخدام:

- success.mp3  → عند الإضافة للسلة (مع fallback لـ beep.mp3 ثم Web Audio)
- error.mp3     → عند خطأ (باركود غير صحيح، كمية غير متوفرة، إلخ)
- checkout.mp3  → عند إنهاء الطلب (Save + Export)

ضع الملفات داخل مجلد public/. يجب أن يكون كل ملف صوتاً مختلفاً (ثلاثة أصوات مميزة).
إن لم يوجد ملف، التطبيق يستخدم نغمة مولدَة (Web Audio) تلقائياً — وكل حدث له نغمة بديلة مختلفة.
