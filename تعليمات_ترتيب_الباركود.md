# تفعيل ترتيب المنتجات حسب الباركود في الصفحة الرئيسية

الترتيب الجديد موجود في الملف **BARCODE_ORDER_NEW.js** لكن التطبيق لا يزال يستخدم **barcodeOrder.js** (الترتيب القديم).

## الحل (اختر أحد الخيارين)

### الخيار 1 — تغيير مصدر الترتيب (الأسرع)

1. افتح الملف **App.jsx**.
2. في السطر 29 ابحث عن:
   ```js
   import { BARCODE_ORDER, sortByBarcodeOrder } from './barcodeOrder';
   ```
3. استبدله بـ:
   ```js
   import { BARCODE_ORDER, sortByBarcodeOrder } from './BARCODE_ORDER_NEW';
   ```
4. احفظ الملف (Ctrl+S).

بعدها ستظهر المنتجات في الصفحة الرئيسية حسب ترتيب الباركود الموجود في **BARCODE_ORDER_NEW.js**.

---

### الخيار 2 — استبدال محتوى barcodeOrder.js

1. افتح **BARCODE_ORDER_NEW.js** وانسخ كل محتواه (Ctrl+A ثم Ctrl+C).
2. افتح **barcodeOrder.js** واستبدل كل محتواه بالمحتوى المنسوخ (Ctrl+A ثم Ctrl+V).
3. احفظ **barcodeOrder.js** (Ctrl+S).

لا تغيّر أي شيء في **App.jsx** في هذه الحالة.

---

بعد تطبيق أحد الخيارين، أعد تشغيل التطبيق أو حدّث الصفحة (F5) لرؤية الترتيب الجديد.
